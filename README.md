# HeliP
Just a simple game written in JAVA which offers fun and exciting gameplay for users.

# Executing Code
Navigate and move into HeliP directory <br />
> cd HeliP <br />

Compile the Code  <br /> 
> javac helip/Game.java <br />
> 
Run <br />
> java helip.Game <br />

Clean up files <br />
rm helip/*.class <br />
