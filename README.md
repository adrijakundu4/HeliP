# HeliP
Just a simple game written in JAVA which offers fun and exciting gameplay for users.

# Executing Code
cd HeliP              | Navigate and move into HeliP directory
javac helip/Game.java | Compile the Code
java helip.Game       | Run
rm helip/*.class      | Clean up files
